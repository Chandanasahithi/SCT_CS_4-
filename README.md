SCT_CS_4 - Advanced Keystroke Logger

🔹 Description

This project demonstrates a basic keystroke logging system developed using Python. The application records user-entered text along with timestamps and stores the logs securely in a text file.

Designed for educational and cybersecurity learning purposes, this project helps in understanding how input monitoring and logging systems work.

⸻

🔹 Features

✔ Records user-entered keystrokes
✔ Stores logs with timestamps
✔ Session-based logging system
✔ File handling using Python
✔ Beginner-friendly cybersecurity project

⸻

🔹 Technologies Used

* Python
* Google Colab
* File Handling

⸻

🔹 How It Works

1. User enters text through the console
2. Program captures the input
3. Current timestamp is generated automatically
4. Logs are stored inside keystrokes.txt
5. Program stops when user enters exit
